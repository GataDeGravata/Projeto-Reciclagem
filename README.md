# Projeto Reciclagem!
### Realização do Projeto APS da Universidade Paulista (UNIP)

Trabalho realizado pelo grupo da Ciência da Computação, terceiro semestre, turma CC3P33.

- Código e interface feitos em Java e JavaFX ☕
- Inspiração e estilo de código encontrados [nesse vídeo aqui](https://www.youtube.com/watch?v=VmChebZcb2U&list=PL2rLmp34yGZwalOzyR96vjOvVIA1b43ws&index=1) ⭐
- Primeiro projeto realizado em Java pelo grupo ❤️

A APS (Atividade Prática Supervisionada) é feito para o aluno adquirir experiências práticas em codificação e projetos reais, sendo avaliados pelo código, empenho, dissertação e organização.

Nesse semestre, a ideia era criar um jogo simples com uma interface gráfica simples, onde o tema seria em torno do meio ambiente e o ecossistema de uma grande metrópole. Após muito tempo discutindo e analisando, decidimos criar um pequeno jogo, onde temos um caminhãozinho coletor de lixo, passeando pela cidade e coletando objetos e materiais para serem reciclados.

Feito em estilo _Snake Game_, o famoso "jogo da cobrinha". Nossa interface gráfica é simples e rudimentar, porém, trás ao jogador o sentimento de nostalgia de um jogo retrô!

Conforme a puntuação e o corpo do caminhãozinho aumentam, a velocidade também irá aumentar, de forma proporcional, tornando cada vez mais desafiador bater seu antigo recorde.
